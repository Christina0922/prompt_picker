# Prompt Picker MVP - 최종 검수 체크리스트

## ✅ 완료된 기능

### 1. 기본 구조 ✓
- [x] Next.js 15 프로젝트 설정 (App Router, TypeScript)
- [x] TailwindCSS 설정
- [x] 폴더 구조 생성 (app, components, lib)
- [x] 환경 변수 설정 (.env.local)

### 2. 핵심 컴포넌트 ✓
- [x] SnippetInput: 조각 입력 텍스트박스
- [x] GoalSelector: 목적 5유형 선택 버튼
- [x] AiSelector: AI 선택 (ChatGPT/Claude/Gemini/자동)
- [x] OptionsPanel: 언어/톤/길이/출력형식 옵션
- [x] PromptCards: 후보 5개 카드 (A~E)
- [x] FinalPromptModal: 최종 프롬프트 확정 모달
- [x] ProGateModal: FREE/PRO 제한 안내 모달

### 3. API 라우트 ✓
- [x] /api/generate: 프롬프트 생성 API
  - 조각 정리 단계
  - 5가지 전략 프롬프트 생성
  - AI별 구조 차별화 (ChatGPT/Claude/Gemini)
  - 길이 옵션 반영 (짧/보통/상세)
  - JSON 응답 강제 + 재시도 로직
- [x] /api/tune: 프롬프트 튜닝 API
  - 더 짧게
  - 더 구체적으로
  - 글자수 지정 (꼬리표 방식)

### 4. 페이지 구현 ✓
- [x] 랜딩 페이지 (/):
  - Hero 섹션

  - 주요 기능 소개
  - 요금제 안내 (FREE/PRO)
- [x] 도구 메인 페이지 (/tool):
  - 모든 입력 컴포넌트 통합
  - 생성 버튼
  - 후보 5개 표시
  - 최종 확정 및 튜닝

### 5. 비즈니스 로직 ✓
- [x] Rate Limiting (localStorage 기반):
  - 하루 3회 제한
  - 24시간 윈도우
  - 제한 초과 시 PRO 모달 표시
- [x] AI별 프롬프트 구조 차별화:
  - ChatGPT: 역할/규칙/출력형식 순서화
  - Claude: 태그 형태 구획 분리
  - Gemini: 제약 앞쪽 고정, 짧고 명확하게
- [x] 5가지 전략 변형:
  - A: 초간단 (최소 지시)
  - B: 균형잡힌 표준
  - C: 제약 강화 (필수/금지 명확)
  - D: 구조 고정 (형식 엄격)
  - E: 확장형 (여러 대안)

## 🎯 검수 항목 (요구사항 대조)

### 필수 기능
| 항목 | 상태 | 비고 |
|------|------|------|
| 조각 입력 → 후보 5개 생성 | ✅ | 정상 작동 |
| 카드 5개가 서로 다른 전략 | ✅ | A~E 전략 차별화 |
| 길이(짧/보통/상세) 반영 | ✅ | promptBuilders.ts에서 처리 |
| AI 선택에 따른 구조 차이 | ✅ | ChatGPT/Claude/Gemini 각기 다름 |
| 최종 1개 확정 → 복사 | ✅ | 모달에서 복사 버튼 |
| 튜닝 버튼 동작 | ✅ | 더 짧게/더 구체적으로/글자수 |
| FREE 3회 제한 | ✅ | localStorage 기반 |
| PRO 안내 모달 | ✅ | 제한 초과 시 표시 |

## 📝 사용 시나리오 테스트

### 시나리오 1: 첫 방문자
1. 랜딩 페이지 확인
2. "무료로 써보기" 클릭
3. 조각 입력 및 옵션 선택
4. 프롬프트 생성 (1/3)
5. 후보 카드 확인 및 선택
6. 최종 프롬프트 복사

### 시나리오 2: 튜닝 사용
1. 프롬프트 생성
2. "이걸로 확정" 클릭
3. "더 짧게" 또는 "더 구체적으로" 선택
4. 튜닝된 결과 확인
5. 복사

### 시나리오 3: FREE 제한
1. 3회 생성 완료
2. 4번째 생성 시도
3. PRO 안내 모달 표시
4. 리셋 시간 안내

## ⚠️ 알려진 제한사항 (MVP 범위)

1. **로그인/회원 관리 없음**: localStorage 기반 제한
2. **실제 결제 연동 없음**: PRO는 UI만 준비
3. **히스토리 저장 없음**: 추후 PRO 기능으로 추가 예정
4. **글자수 정밀 지정**: PRO 전용으로 잠금 (MVP에서는 꼬리표 방식)
